iOS Events App - developed for Mobile & Web Technologies I module

- Uses data from Eventful API to display events. 
- Users can search for any event in the Eventful database and 
  save it to the app. They can also save it to the iOS calendar
  and post about it on Facebook and Twitter.
- Selecting events URL will send users to Eventful to purchase
  tickets to the event.
- Custom events can be created by users.

Written in Swift. App is required to be opened in XCode IDE in order 
to run either on an iPhone/iPad or an emulator. An internet 
connection is required for the application to work correctly.
